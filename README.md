**ProxyKit is a local-first API traffic workspace for capturing, inspecting, replaying, modifying, mocking, fuzzing, testing, and understanding real app traffic.**

ProxyKit captures real app traffic and turns it into replayable requests, mocks, regression tests, sanitized reports, and privacy/security findings.

It combines a desktop MITM proxy, live traffic inspector, full-featured API client, replay workspace with load testing, mock server, rewrite engine, fuzzer, breakpoints, scripts, chaos testing, session reports, privacy analysis, 24-dimension analysis workspace, and optional team cloud controls into one tool.

ProxyKit is built for developers, QA engineers, mobile teams, and platform teams who need to debug what an app actually sends over the network, not only hand-crafted API requests.
